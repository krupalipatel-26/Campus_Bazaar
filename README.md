## 📖 About the Project

CampusBazaar is a responsive e-commerce inspired web application designed exclusively for college students. It provides a platform where students can buy, sell, and exchange college essentials such as books, laptops, scientific calculators, bicycles, stationery, and hostel accessories.

This project was developed as part of my **Web Development Internship** using **HTML, CSS, and JavaScript** while following modern front-end development practices.
